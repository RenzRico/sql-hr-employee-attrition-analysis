HR Employee Attrition Analysis (SQL Project)
📌 Project Overview
This project analyzes an HR employee dataset using SQL to identify
patterns related to employee attrition, compensation, and workforce
distribution.
The goal is to demonstrate practical SQL skills applied to
real-world HR and People Analytics scenarios.

🗂 Dataset

Source: Kaggle – HR Analytics (Employee Attrition) Dataset
Records: Employee demographic, job, and compensation data
Granularity: One record per employee


🛠 Tools Used

SQL (MySQL compatible)
GitHub
Kaggle Dataset


📊 Key Business Questions Answered

What is the overall employee attrition rate?
Which departments have the highest attrition?
Which job roles experience the most employee turnover?
How does salary differ between employees who stayed vs left?
Does employee tenure impact attrition?
How do job satisfaction and work-life balance relate to attrition?


🧠 SQL Skills Demonstrated

SELECT, WHERE, ORDER BY
GROUP BY, HAVING
CASE WHEN statements
Subqueries
Aggregate functions
Basic KPI calculations (attrition rate, averages)


📈 Sample Insights

Certain departments show higher attrition rates than others.
Employees with lower tenure are more likely to leave.
Lower job satisfaction is associated with higher attrition.
Compensation levels vary significantly by job role and department.


✅ How to Use

Download the dataset from Kaggle
Import the CSV file into your SQL database
Run the queries from the /queries folder
Review insights and results
